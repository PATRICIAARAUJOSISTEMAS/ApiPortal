# ApiPortal
Api .Net Core

Language C# 4.6.2
.Net Core2.0
Auth0 Authentication and Authorization
Entity Framework
Sql Server (mssql-server-linux)
Docker
Documentação da Api com Swagger - acesso ao swagger - http://localhost:5000/swagger/
Teste Unitário. 
OBS.: Swagger implementa o Curl.
